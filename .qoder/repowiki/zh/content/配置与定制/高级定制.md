## 详细组件分析

### USER_PROMPT_PRESETS 预设场景详解与使用
- 预设集合位置：共享配置中定义了多类场景的 user prompt 模板，便于快速选择
- 场景与要点
  - general：通用结构化解析，适合大多数图像理解任务
  - photo：强调摄影技术参数（光质、色温、对比度、镜头特性、景深等）
  - cg：关注数字制作与渲染管线（光照、材质、风格影响等）
  - design：面向平面设计的信息架构与视觉系统（排版、网格、负空间、设计运动等）
  - ui：界面架构与设计系统（组件库特征、导航模式、交互暗示、设计令牌、界面风格）
  - assets3d：3D 资产拓扑与材质逻辑（PBR 映射、网格密度、表面瑕疵等）
  - product：电商产品拍摄与英雄图（灯光布置、材质质感、品牌细节、环境心理定位）
  - anime：动漫插画角色设计与风格特征（角色类型、表情、发型、服装、特效等）
  - architecture：建筑室内设计的空间布局与结构特征（建筑风格、材料、光线设计等）
  - food：美食摄影的质感表现与摆盘技巧（食材质感、光线、色彩搭配等）
  - portrait：人像摄影的面部特征与拍摄技巧（表情、光线、化妆、服装等）
- 使用方式
  - 在选项页面的"提示词"区域点击对应预设按钮，即可自动填充 user prompt
  - 若需微调，可在文本域直接编辑，随后自动保存

**更新** 新增了 anime、architecture、food、portrait 等专业领域模板，丰富了预设场景

### 高保真重建模式配置选项
- 配置位置：在 DEFAULT_SETTINGS 中新增 recreateMode 选项，默认为 false
- 功能说明：启用后将使用更详细的分析提示词，生成更精确的还原提示词（包含负面提示词和技术参数）
- 核心增强功能
  - 系统提示词构建机制：RECREATE_MODE_SYSTEM_OVERLAY 提供专业级重建约束
  - 用户提示词增强逻辑：RECREATE_MODE_USER_OVERLAY 添加超详细重建分析要求
  - 负向提示词生成：自动生成英文负向提示词，防止漂移错误
  - 技术参数汇总：parameters 字段总结重建控制参数（主体比例、布局锁定、材质锁定等）
- 启用方式：在选项页面的"使用体验"部分勾选"高还原度模式"开关
- 适用场景：需要极高精度图像还原的专业场景，如产品摄影、艺术创作、设计复原、动漫制作等

**更新** 高保真重建模式得到重大增强，新增了专业级的系统提示词构建机制和负向提示词生成功能

### 自定义提示词模板：创建、命名、保存与删除
- 创建与命名
  - 点击"添加自定义"按钮进入自定义模式

  - 保存后会在预设区出现带"⚙️"前缀的自定义芯片，点击即可加载

- 数据持久化
  - 所有设置与自定义模板统一保存在本地存储，变更后自动广播给内容脚本以即时生效

### UI_STRINGS 与 SETTINGS_I18N 国际化机制
- UI_STRINGS：内容脚本与后台服务用于面板状态、按钮文案、错误提示等 UI 文本的本地化字典
- SETTINGS_I18N：选项页面的静态文案与占位符，配合语言切换实现设置面板的双语显示
- 运行机制
  - 选项页面加载时根据 uiLanguage 应用 SETTINGS_I18N 中的文案
  - 内容脚本与后台服务根据当前语言从 UI_STRINGS 读取对应文本
  - 设置变更后通过广播通知其他模块同步更新
- 调整方法
  - 在选项页面的语言切换控件中选择 zh 或 en
  - 对应的 data-i18n 属性会自动替换为 SETTINGS_I18N 中的翻译
  - 主面板内的动态状态文本也会基于 UI_STRINGS 进行翻译

### 高级定制技巧与最佳实践
- 预设优先，模板补充
  - 优先选用贴近场景的预设作为基础，再针对具体项目微调 user prompt
  - 将高频使用的定制模板命名为易识别的标题，便于快速复用
- 高保真重建模式应用
  - 在需要极高精度还原的场景下启用高保真重建模式
  - 结合专业的摄影、设计、艺术、动漫等领域模板获得更精确的结果
  - 利用负向提示词和参数汇总功能提升重建质量
- 语言与界面一致性
  - 统一设置面板语言与主面板语言，避免混用造成阅读负担
  - 当需要与团队协作时，建议固定语言并共享自定义模板
- 历史记录管理
  - 定期清理历史记录，减少本地存储占用
  - 复杂项目可导出提示词后归档，再在扩展中复用
- 性能与兼容性
  - 遇到 400 错误时，适当降低图片分辨率限制
  - 不同模型对图片输入格式敏感，必要时切换请求格式或模型
- 快捷键与悬浮入口
  - 启用悬浮入口与截图快捷键，提升批量处理效率
  - 在复杂页面中合理关闭悬浮入口，避免干扰

## 故障排查指南
- 常见错误与定位
  - 认证失败/权限不足：检查 API 密钥与权限
  - 调用次数超限：降低频率或升级配额
  - 服务器错误/超时：降低分辨率或更换模型
  - 模型返回非 JSON：调整 system prompt 确保输出纯 JSON
  - 缺少 zh/en 字段：确保 system prompt 正确生成双语字段
- 用户可见提示
  - 后台与内容脚本均基于 UI_STRINGS 提供用户友好提示
  - 错误码与错误消息在 ERROR_CODES 与 ERROR_MESSAGES 中定义
- 处理建议
  - 优先核对连接设置与模型配置
  - 使用历史记录回放定位问题
  - 切换语言与分辨率进行对比测试

## 结论
通过 USER_PROMPT_PRESETS 的场景化模板、高保真重建模式配置选项、自定义模板的灵活扩展、UI_STRINGS 与 SETTINGS_I18N 的双语支持，以及后台与内容脚本的协同，Img2Prompt 提供了专业级的定制能力。本次更新大幅增强了高保真重建模式的功能，新增了系统提示词构建机制、用户提示词增强逻辑、负向提示词生成、技术参数汇总等专业级功能，显著提升了图像还原的精确度和实用性。相比之前的简化版本，本版本专注于提供全面的高级定制能力，满足专业用户的需求。结合最佳实践与故障排查方法，用户可以按自身工作流高效地生成与复用高质量提示词，并在多语言环境下保持一致的使用体验。
